# PayOne Java SDK

This repository contains the PayOne PCP Server SDK for Java.  
An example App is also provided in the `app` directory.

### Run the example app

```sh
API_KEY=api_key API_SECRET=api_secret ./gradlew :app:run
```

### Build the library

```sh
./gradlew :lib:build
```
